# Tale of Immortal Tool (toi)

A modding utility for the [鬼谷八荒 Tale of Immortal][steam-toi].

Mainly supports the following features:
- Encrypting and decrypting mod files, or any files encrypted using the game's `EncryptTool` API.
- Packing and unpacking mods: packed mods can be loaded by the game, while unpacked mods cannot (similar to cooked vs. uncooked mods).
- Packing and unpacking savegames (PC saves only).

[steam-toi]: https://store.steampowered.com/app/1468810/_Tale_of_Immortal/
